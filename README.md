# Tutorial-Board
1. 개발 환경 
   - Intellij Ultimate
   - Myql 8.0.22.0
   - JDK 1.8
2. 개발 기간
   - 14일
3. 사용 테이블
   - Member: 회원정보를 저장하기 위한 테이블 
   - Contents: 게시글의 정보를 저장하기 위한 테이블 
   - ContentsItems: MemberId와 ContentsId를 저장하기 위한 테이블 
   - Files: 파일정보를 저장하기 위한 테이블
4. 핵심 기능
   - JPA, MVC, Spring Security
5. 구현 기능
   - 회원등록, 로그인, 게시글 등록 & 수정 & 삭제, 파일 업로드 & 다운로드
